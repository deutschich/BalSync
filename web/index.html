Commands & Usage

/balsync reload

Reloads the plugin's configuration file without needing a server restart.

/balsync save

Forces an immediate save of all player balances from server memory to the database.

/balsync load

Forces an immediate load of all player balances from the database into server memory.

# Permissions: balsync.admin

Configuration

BalSync is highly configurable. Here's the default config.yml structure:

# Database Configuration

database:

  host: "localhost"

  port: 3306

  database: "minecraft"

  username: "root"

  password: "password"

  use-ssl: false


# Plugin Settings

settings:

  # Auto-save interval in seconds (0 to disable)

  auto-save-interval: 60

  # Whether to save on player quit

  save-on-quit: true

  starting-balance: 100.0

  locale: "en"
